unboxing

1. You normally get two boxes:
- one little box with batteries
- one big box with the electronics and the remote control of the drone
If this is not the case, come see your makerslab manager asap!


2. The little box contains:
- one 9V battery for the remote control
- two LiPo batteries 3.7V for the drone. (A USB cable in the other box will allow you to recharge these batteries)


3. The big box contains:
- the remote control (arduino UNO + shield)
- the joysticks of the remote control
- a mini RF emiter shield to allow the remote control to communicate with the drone
- a blue USB cable to upload the code of the remote control
You can already put the mini RF emiter shield in place as well as the two joysticks.

- the drone's hunt (based on an arduino mini)
- a mini RF receiver shield
- 3 propellers' set (to be able to change them in case of crash)
- 2 motors' set (in case you need to change them)
You can already put the RF receiver mini shield in place.

- a USB-Serial mini shied to modify the settings of the drone (only for more advanced)
- a micro USB cable to connect this mini shield to your computer
Keep carefully these two elements.

Before going any further, you will have to design and manufacture the drone arms & propeller protections.
